# cubeviz

A Cube Vizualizer.

## Usage

```bash
$ cubeviz <input> > <out.svg>
```

## Input Examples

A Face 3x3

```
Face {
  O O W
  W W Y
  Y Y B
}
```

A Face with its side

```
Face {
    G G B
  W O O W R
  B W W Y O
  G Y Y B Y
    O G R
}
```

## Colors

- `W`, White
- `Y`, Yello
- `R`, Red
- `O`, O
- `B`, Blue
- `G`, Green
- `.`, Masked cube
